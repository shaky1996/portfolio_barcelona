# Portfolio Project - FC Barcelona

For live demo, please click [here](https://shakbarcelona2024.web.app/)

Website was deployed through Google's Firebase. 

## Project Description

This project is a responsive website dedicated to FC Barcelona, one of the most iconic football clubs in the world. Built using fundamental web technologies such as HTML, CSS, and JavaScript, this website serves as a showcase of skills and knowledge in web development without relying on frameworks.

## :white_check_mark:	Features

- **Responsive Design:** The website is designed to provide an optimal viewing experience across a wide range of devices, from desktops to smartphones.
- **Live Data Integration:** Utilizing APIs, the website fetches real-time information including news updates, upcoming matches, recent results, and current standings of FC Barcelona in Spanish Football League (La Liga).
- **News Section:** Stay updated with the latest news and articles related to Spanish Footbal League (La Liga).
- **Next Game:** Get information about the next match, including date, time, location and opponent.
- **Results:** View the most recent results of FC Barcelona match.
- **Standings:** Check the current standings of FC Barcelona in Spansih Football league (La Liga).


## :white_check_mark:	Technologies Used

**HTML**: Markup language for structuring the content of the website.
**CSS**: Styling language for designing the layout and appearance of the website.
**JavaScript**: Programming language used to enhance interactivity and dynamic content.
**API Integration**: Utilizes APIs to fetch and display live data about FC Barcelona.

## :white_check_mark:	Motivation

The primary motivation behind this project is to demonstrate proficiency in web development using core technologies—HTML, CSS, and JavaScript — without relying on frameworks. By building a website dedicated to FC Barcelona, it also serves as a platform to showcase passion for football and web development skills simultaneously.
